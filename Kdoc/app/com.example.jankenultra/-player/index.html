És una classe intermedia utilizada para crear objetos que seran utilizados por otra classe para mostrar los ejercicios y sus datos

Parameters

nomExercise

nombre del ejercicio

nReplays

numero de repeticiones por serie

nSeries

numero de series que se deben hacer

rest

tiempo de descanso entre series

suggestion

consejo o descripción de como hacer el ejercicio

complete

para saber si han hecho el ejercicio o no

imgId

imagen de ayuda para saber como hacer el ejercicio

pathing

recorrido donde se encuentran los ejercicios en el Firebase.
